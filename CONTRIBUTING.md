# Contributing to p5.videoeditor.js

First off, thank you for considering contributing to p5.videoeditor.js! Your help is greatly appreciated.

## How Can I Contribute?

### Reporting Bugs
If you find a bug, please make sure it hasn't already been reported by searching the existing issues. If you can't find an existing issue, please open a new one using the "Bug report" template.

Please include as many details as possible in your bug report.

### Suggesting Enhancements
If you have an idea for a new feature or an improvement to an existing one, please open an issue using the "Feature request" template. This allows for a discussion with the community about the proposed feature.

### Submitting Code Changes
If you'd like to contribute code, please follow this workflow:

1.  **Fork the repository:** Create your own fork of the project on GitHub.
2.  **Create a new branch:** Make a new branch from the `develop` branch for your feature or bug fix.
    ```bash
    git checkout -b feature/your-amazing-feature
    ```
3.  **Make your changes:** Write your code and add any necessary tests.
4.  **Ensure tests pass:** Run the test suite to make sure your changes haven't broken anything.
5.  **Submit a Pull Request:** Push your branch to your fork and open a pull request against the `develop` branch of the main repository.

Thank you for your contribution!
